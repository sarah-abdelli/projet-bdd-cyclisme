# Base de données & Application Web - Palmarès cyclisme

## Description

Ce projet consiste à concevoir une base de données et développer une application web permettant de consulter le palmarès des grands tours cyclistes.

## Fonctionnalités

* Consultation des coureurs
* Affichage des éditions des courses
* Classements des coureurs
* Interaction avec la base de données via PHP

## Structure du projet

* /sql : scripts SQL (MySQL, PostgreSQL)
* /web : application web (HTML, CSS, PHP)
* /docs : rapport du projet

## Technologies utilisées

* SQL (MySQL, PostgreSQL)
* PHP
* HTML / CSS

## Compétences développées

* Modélisation de base de données (MCD → relationnel)
* Requêtes SQL
* Connexion base de données avec PHP
* Développement web
